MIS 311 - Data Analysis Project

Project Title
Most Visited Countries Analysis

Project Description
This project analyses international tourist arrivals across different countries using descriptive statistics and data visualisation techniques.

Tools Used
- Microsoft word
- Microsoft Excel
- Descriptive Statistics
- Box and Whisker Plot

Key Insights
1. The international tourism industry has recovered strongly and continues to maintain a stable growth trend.
2. The distribution of international tourists is uneven, with most being concentrated in popular destinations around the world.

This repository included
- Dataset
- Data analysis report
- Visualisation chart
